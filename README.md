# krsHubSwap - Labymod 3 Addon (Fixed)

Оригинальный Fabric мод переработан в аддон для Labymod 3.

## Описание

Аддон для быстрого телепортирования на разные лайт анархии сервера.

## Команды

По умолчанию доступные алиасы: `lt`, `an`

```
/lt <номер>     - телепортироваться на лайт анархию
/an <номер>     - альтернативный алиас
```

## Настройки

Откройте конфиг Labymod и найдите настройки krsHubSwap для:
- Установки алиасов команд
- Настройки диапазонов анархий (Соло, Дуо, Трио, Кланы)
- Общего количества анархий на сервере

## Что изменено в этой версии

✅ Исправлены конструкторы `RangeConfig` и `AliasConfig` - изменены с `private` на `public`
✅ Проект теперь собирается без ошибок

## Сборка

### Windows
```bash
build.bat
```

### Linux/Mac
```bash
bash build.sh
```

### Gradle
```bash
gradle fatJar
```

JAR файл будет в папке `build/libs/`

## Установка

1. Собери проект (см. выше)
2. Скопируй JAR в папку `.minecraft/labymod/addons/`
3. Перезагрузи игру
